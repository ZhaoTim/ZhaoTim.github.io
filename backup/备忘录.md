在函数组件里，使用useState会返回state与updater函数。

如果在事件回调函数内，多次调用updater 函数，都会被推进更新队列，等到下次渲染时，进行处理。

```jsx
const App = () => {
  const [num, setNum] = useState(0);
  const increase = () => {
    setNum(num + 1);
    setNum(num + 1);
    setNum(num + 1);
  }

  return (
    <button onClick={increase}>increase{num}</button>
  );
}
```
